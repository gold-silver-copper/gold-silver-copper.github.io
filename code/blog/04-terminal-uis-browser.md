# Terminal UIs in the Browser

*2025-04-20*

This website is built entirely with Ratzilla, which brings
Ratatui's terminal UI framework to the browser via WASM.

**TachyonFX** adds shader-like visual effects — the background
animation, page transitions, and link click effects are all
powered by tachyonfx running in WebAssembly.

No JavaScript framework. No DOM manipulation. Just Rust
rendering a terminal buffer to a canvas element.
